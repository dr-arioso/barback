# Glossary

**Resolver**  
A module that converts an image into structured item data.

**Output Plugin**  
A module that exports results into a specific format.

**Item**  
The normalized representation of a recognized product.

**StashKit**  
A future standalone library that contains resolver APIs, output APIs, and item processing.

**Barback**  
A CLI wrapper around StashKit, designed for batch processing of product images.
