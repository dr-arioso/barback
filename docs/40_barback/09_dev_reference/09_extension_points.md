
# Extension Points

Developers can extend Barback via:

## 1. Custom Skills
Add new signal sources (e.g., web scrapers, LLM inference, chemical analysis).

## 2. Custom Resolvers
Specialized flows (e.g., beer-only resolvers).

## 3. Custom Pipelines
Alternative or multi-step pipelines (multi-image pipelines).

## 4. Custom Backends
Encrypted storage, cloud sync, POS integration.

## 5. Custom Booster Packs
Domain packs for non-booze items (tea, spices, coffee beans, cigars, etc.).

Each extension point has stable interface definitions.
