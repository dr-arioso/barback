
# Cocktail Roles StashTags

These tags describe how an ingredient behaves in cocktail construction.

```
cocktail_roles:
  base_spirit
  modifier
  lengthener
  sour_agent
  sweetener
  foaming_agent
```

Guidelines:
- Base spirit should apply only to spirits typically used as the foundation.
- Modifier covers amari, liqueurs, vermouths, syrups, juices.
- Lengthener includes soda, tonic, sparkling wine.
