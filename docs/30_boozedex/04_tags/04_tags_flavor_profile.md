
# Flavor Profile StashTags

Flavor tags describe sensory attributes.

```
flavor:
  citrus
  herbal
  floral
  spicy
  smoky
  woody
  earthy
  bitter
  sweet
  nutty
  fruity
  berry
  stone_fruit
  tropical
```

Notes:
- Tags can combine (e.g., herbal + bitter for Campari).
- Subtags (berry, stone_fruit) allow fine-grained filtering.
