
# Flavor Profile Overlay

Flavor profile tags are used to classify tasting characteristics.

```
flavor_profiles:
  citrus:
  herbal:
  floral:
  spicy:
  smoky:
  woody:
  earthy:
  bitter:
  sweet:
  nutty:
  fruity:
    stone_fruit:
    berry:
```

Examples:
- Mezcal → smoky  
- Campari → bitter / citrus  
- Amaretto → sweet / nutty
