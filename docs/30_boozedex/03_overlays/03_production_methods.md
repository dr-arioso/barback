
# Production Method Overlay

Production method tags add specificity to spirits and wines.

```
production_methods:
  pot_still:
  column_still:
  blended:
  barrel_aged:
  wild_ferment:
  filtered:
  charcoal_filtered:
  macerated:
```

Examples:
- Tennessee whiskey → charcoal_filtered  
- Cognac → pot_still / barrel_aged  
- Natural wine → wild_ferment
