
# Skill Examples

## 1. UPC Recognition (Image → String)
Uses computer vision to detect UPC/EAN codes.

## 2. OCR Extraction (Image → Text)
Extracts printed text from labels.

## 3. Manual Entry Skill
Prompts the user for keyboard or voice input.

## 4. AI Vision Skill
Uses LLM or Vision API to extract product descriptors.

## 5. Classification Skill
Maps descriptors to DexNodes using BoozeDex taxonomy.
