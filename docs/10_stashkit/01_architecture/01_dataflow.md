
# StashKit Dataflow

1. **Input arrives** (image, text, UPC, product name).
2. **Resolvers** select Skills to process the input.
3. **Skills** produce structured results with confidence.
4. **Resolvers** merge/conflict-resolve outcomes.
5. **Taxonomy lookup** assigns classification using DexNodes.
6. **Models** are constructed and validated.
7. **Stashes** persist objects with indexing and metadata.
