
# StashKit Subsystems

## 1. Taxonomy Subsystem (StashDex)
Provides hierarchical category structures using DexNodes and metadata via StashTags.

## 2. Skills Subsystem
Defines atomic units of inference with consistent confidence scoring and structured outputs.

## 3. Resolver Subsystem
Coordinates Skills, manages retries, fallback logic, conflict merging, and produces validated domain models.

## 4. Stashes Subsystem
Typed storage engine with indexing, lookup, versioning, and optional persistence backends.

## 5. Booster Subsystem
Loads and integrates DexPacks, TagPacks, SkillsPacks, ResolverPacks, and StashPacks.
