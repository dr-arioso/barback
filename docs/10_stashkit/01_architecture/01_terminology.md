
# Terminology

- **DexNode**: Node in a taxonomy tree.
- **DexPack**: BoosterPack providing taxonomy.
- **TagPack**: BoosterPack providing metadata overlays.
- **SkillsPack**: Package containing Skills.
- **ResolverPack**: Package containing Resolvers.
- **StashPack**: Domain bundle combining multiple boosters.
- **StashTags**: Metadata attached to DexNodes.
