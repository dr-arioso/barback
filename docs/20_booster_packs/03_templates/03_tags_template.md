
# Template: StashTags Definition

Example `cocktail_roles.yaml`:

```
id: cocktail_roles
version: 1.0
type: tags

tags:
  base_spirit:
    description: Primary alcohol in a cocktail
  modifier:
    description: Flavor or sweetness-modifying ingredient
  lengthener:
    description: Adds volume without raising ABV
  garnish:
    description: A non-liquid decorative component
```

Guidelines:
- Tags should be human-readable.
- IDs should be stable and globally unique.
