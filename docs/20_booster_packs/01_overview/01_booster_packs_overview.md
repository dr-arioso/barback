
# Booster Packs Overview

**Booster Packs** are modular, installable extensions that add domain-specific
intelligence to StashKit. They bundle:

- Taxonomies (DexTrees)
- StashTags
- Default Skills
- Default Resolvers
- Default Pipelines
- Default Configuration
- Optional Persistence templates
- Optional metadata overlays

A Booster Pack transforms StashKit from a generic inference framework into a
domain-ready system (e.g., Barback for beverages).

Booster Packs are designed to be:

- **Compositional**  
- **Declarative**  
- **Easily versioned**  
- **Override-friendly**
