
# Booster Pack Examples

## Example 1: BoozeDexPack
Adds:
- Booze taxonomy (spirits, wine, beer, liqueurs, mixers)
- Cocktail roles
- Flavor tags
- BottleResolver defaults
- BottlePipeline defaults
- UPCSkill configuration
- ClassificationSkill aware of BoozeDex

## Example 2: HardwarePack
Adds:
- Tool taxonomy
- Part classification
- OCR rules for tool labels
- Resolver for part numbers

## Example 3: IngredientPack
Adds:
- Food taxonomy
- Unit conversion skill
- Allergy metadata
- Nutrition model
