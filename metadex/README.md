# MetaDex

Canonical ontology and compiler for the StashKit / Barback architecture.

This repo contains:

- `source_yaml/` — human-editable ontology (entities, relationships, rules)
- `compiler/` — MetaDex compiler (v1.2, with YAML normalization)
- `metadex/metadex.json` — compiled ontology (placeholder, regenerate via compiler)
- `boosterpack/` — Developer BoosterPack using the compiled MetaDex
- `docs/` — documentation for humans and for GPT onboarding
- `tests/` — test scaffolding for the compiler and ontology
